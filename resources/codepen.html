CodePen ist eine Online-Plattform mit Elementen eines sozialen Netzwerks für Frontend-Webentwickler, die ihre Arbeit präsentieren möchten, kleine Code Snippets mit anderen teilen wollen oder einfach nur auf der Suche nach Inspiration sind.

Das Konzept ist relativ simpel: Man erstellt sich einen Account und bekommt eine eigene Profilseite, die ein wenig an ein Portfolio erinnert. Anschließend kann man über den Online-Editor von CodePen sogenannte Pens erstellen, die im Grunde für kleine Code Snippets bis hin zu One-Page-Websites vielfältig einsetzbar sind und die Nutzung von HTML, CSS, JavaScript sowie verschiedene Präprozessoren, Frameworks und sonstige hilfreiche Dinge für die Entwicklung erlauben. Diese Pens erscheinen dann auf der eigenen Profilseite, sodass man sich super eine Art Portfolio mit CodePen aufbauen kann.

Damit ist aber noch lange nicht Schluss. Neben den Pens bietet CodePen auch noch die Erstellung und Verwaltung von ganzen Projekten und das Verfassen von Artikeln an, die ebenfalls auf der Plattform erstellt und veröffentlicht werden können.

Die grundsätzlichen Funktion auf der Plattform sind dabei immer kostenlos. Möchte man jedoch ein paar mehr Funktionen genießen können, gibt es noch die Möglichkeit, ein Upgrade des Accounts auf Pro vorzunehmen. Für die Meisten sollte der kostenlose Account aber reichen. Ich komme damit beispielsweise auch ohne Probleme aus.
