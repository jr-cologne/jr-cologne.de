# jr-cologne.de

The source code of my [personal website](https://jr-cologne.de/), which will go online in September.

Currently, the website is in version 0.6.1. You can see the site in action under [dev.jr-cologne.de](https://dev.jr-cologne.de/).

## Development

The development of my website started on the 24th July 2017 and ended on the 28th August 2017. That does not mean that it is completely finished, but the focus is now on the content of the website, and if any, I only make small changes to the website itself.
